# AutoSplitVideo
Bilibili 直播录制和开播提醒

# 主要功能
- [x] 多直播间录制
- [x] 支持录制付费直播
- [x] 开播提醒
- [x] FLV 文件修复、处理
- [x] 记录直播间标题变化
- [ ] 录制实现很傻，直接 http 下载，有空再写 flv 实现吧
- [ ] 速度监控还没做，如果网络环境差录制可能卡住
